Cara Survive di Teknik Informatika 🎀

📚 1. Kenalan Sama Dosen & Teman

Networking itu penting! Jangan malu buat ngobrol, nanya tugas, atau ikut project bareng. Atau yaa... minimal jangan musuhan lah, itu aja udah effort 😅

💻 2. Rajin Ngoding Walau Error

Error itu teman! Biasain latihan dari hal kecil kayak kalkulator sampai project mini kayak website portfolio lucu. Tapi jangan heran ya, compiler itu suka ngeselin, bahkan kalau kamu ngetik "Hello World" pun bisa error ✨

🧠 3. Pahami Konsep Dasar

Struktur data, algoritma, dan logika pemrograman itu fondasi. Jangan cuma hafal, coba buat analogi ala kamu sendiri! Misalnya queue itu kayak antre beli boba—first come, first served. Tapi kadang ya... dosennya explain-nya kayak lagi bahas kuantum 🤡

🌸 4. Jangan Lupa Istirahat dan Skincare!

Kesehatan mental itu penting. Coding boleh, tapi me time juga harus. Pakai masker, journaling, nonton anime, go for it! Walau deadline udah kayak mantan yang nggak move on, tetep harus glowing dong~

🌐 5. Ikut Komunitas & Event TI

Biar nggak merasa sendirian. Banyak komunitas yang seru dan ngasih insight, bahkan lowongan magang! Walaupun awalnya dateng cuma buat stiker gratisan dan snack, it's okay bestie 😇

🛍️ 6. Gunakan Tools Lucu & Bermanfaat

VS Code bisa kamu buat aesthetic pake tema pink, font lucu, dan plugin membantu. Productive & cute! Kalau hidupmu chaos, setidaknya editor kamu rapih 💅

🧋 7. Jangan Lupa Ngopi (Atau Boba)

Tidur jam 3, ngoding jam 4, ujian jam 7. Hidupmu mungkin chaos, tapi minimal hatimu hangat... berkat kopi
